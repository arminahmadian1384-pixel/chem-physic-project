آموزش و راهنما

این سایت برای کمک به یادگیری شیمی فیزیک طراحی شده است.

  • ΔG, ΔH, ΔS: تغییرات انرژی واکنش و امکان خودبه‌خودی بودن واکنش
  • واکنش خودبه‌خودی: اگر ΔG < 0 → خودبه‌خودی، اگر ΔG > 0 → غیرخودبه‌خودی
  • واحدها: ΔH → kJ/mol, ΔS → J/mol·K, ΔG → J/mol
  • نمودار ΔG با دما: نشان می‌دهد در چه دماهایی واکنش خودبه‌خودی است
  • شبیه‌ساز کینتیک: نحوه تغییر غلظت با زمان را نمایش می‌دهد
  • ماشین‌حساب گرما: محاسبه q = m·c·ΔT
